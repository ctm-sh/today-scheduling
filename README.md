# 오늘 할일

타임라인 기반의 싱글 페이지 할 일 관리 웹앱입니다. `index.html` 파일 하나로 동작하며, 데이터는 브라우저의 `localStorage`에 저장되어 별도의 서버나 설치 과정이 필요하지 않습니다.

## 개요

"오늘 할일"은 **하루 단위의 시간 계획**과 **할 일 목록 관리**를 하나의 화면에서 처리할 수 있도록 설계된 경량 생산성 도구입니다. 시간대를 드래그하여 일정 블록을 만들고, 카테고리와 색상으로 분류하며, 뽀모도로 타이머로 실행을 돕고, 주간·월간 통계로 패턴을 되돌아볼 수 있습니다.

- **형태**: 정적 HTML 한 장 (외부 의존성은 CDN에서 로드되는 `canvas-confetti`, `html2canvas`, Google Fonts)
- **실행 방법**: `index.html`을 브라우저에서 열기
- **데이터 저장**: 브라우저 `localStorage` (JSON 내보내기/가져오기로 백업 가능)
- **언어**: 한국어 UI

## 주요 기능

### 일정 관리
- **타임라인 뷰**: 시간 그리드를 드래그하여 할 일 블록을 직접 생성·편집·삭제
- **할 일 목록 뷰**: Enter 키로 빠른 추가가 가능한 리스트 형식
- **시간 미정 항목**: 특정 시간대 없이 등록한 할 일을 별도 섹션에서 관리
- **날짜 이동**: 헤더의 이전/다음 버튼으로 날짜별 기록 조회
- **현재 시각 표시**: 타임라인에 실시간 라인으로 표시

### 분류 및 진행 관리
- **카테고리 관리**: 사용자 정의 카테고리와 컬러 팔레트
- **진행률 표시**: 완료/전체 건수와 퍼센티지를 헤더 바에 표시
- **한줄 회고**: 하루를 마무리하는 간단한 텍스트 메모 저장

### 뽀모도로 타이머
- **독립 실행 가능**: 헤더의 타이머 아이콘으로 언제든 실행 — 할 일을 등록하지 않아도 바로 사용
- **할 일과 연동(선택)**: 타임라인의 각 할 일에 있는 ▶ 버튼을 누르면 해당 할 일을 대상으로 시작
- **집중 시간 직접 설정**: 기본값 25분, 모달에서 1~120분 사이로 변경 가능(설정값은 브라우저에 저장)
- **집중 / 휴식 자동 전환**: 집중 단계 종료 후 5분 휴식, 이어서 다시 집중 단계로 순환
- **시작 / 일시정지 / 재개 / 리셋 / 종료** 제어 제공
- **헤더 미니 위젯**: 실행 중 남은 시간이 헤더에 표시되며 클릭 시 전체 모달 열기
- **단계 전환 알림**: 집중·휴식 종료 시 브라우저 알림으로 전환 시점을 안내

### 집중 도구
- **완료 이펙트**: 할 일 완료 시 confetti 애니메이션

### 통계 및 백업
- **주간/월간 대시보드**: 카테고리별 시간 분포와 완료 패턴 시각화
- **PNG 내보내기**: 오늘의 스케줄을 이미지로 캡처 (`html2canvas`)
- **JSON 내보내기/가져오기**: 전체 데이터 백업 및 복원

### UI/UX
- **다크/라이트 테마 전환**
- **모바일 대응**: 타임라인·목록 탭을 전환하는 모바일 레이아웃
- **키보드/드래그 중심 조작**: FAB 버튼, Enter 단축, 시간대 드래그 선택
